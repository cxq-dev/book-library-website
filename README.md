# book-library-website

一个基于 HTML / CSS / JavaScript 开发的**教材图书馆静态网站**，模拟了前端教材的在线展示与筛选功能，适合作为前端入门项目。

---

## ✨ 项目功能
- 教材列表展示：清晰呈现书籍编号、书名、价格、所属课程与封面图片
- 分类筛选：支持按课程类型（HTML / JavaScript / Vue）快速过滤教材
- 详情预览：点击封面可查看书籍出版社、折扣价等详细信息
- 响应式布局：使用 Flex 布局搭建页面结构，适配不同屏幕尺寸

---

## 🛠️ 技术栈
- **HTML5**：搭建语义化页面结构
- **CSS3**：Flex 布局 + 基础样式美化，实现页面排版与视觉效果
- **JavaScript**：实现分类筛选、交互逻辑与数据渲染
